# Project Dependencies

This document lists all major software libraries, frameworks, and tools required to run the Voice-Based POS Billing System.

---

## Frontend Dependencies

### Core Framework
- React (Vite)

### HTTP Client
- Axios

### Styling
- Tailwind CSS

### Icons
- Lucide React

### Charts / Analytics
- Chart.js
- React ChartJS 2

### Routing
- React Router DOM

---

## Backend Dependencies

### Runtime
- Node.js

### Web Framework
- Express.js

### Database
- MongoDB
- Mongoose

### Environment Management
- Dotenv

### Cross-Origin Requests
- CORS

### Development Utilities
- Nodemon

---

## Speech Recognition

- Web Speech API (Browser-based speech recognition)

---

## Evaluation & Research Utilities

- CSV Parser
- File System (Node.js)
- High Resolution Timer (Performance API)

---

## Deployment Tools

- Git & GitHub
- Vercel (Frontend Hosting)
- Render / Railway (Backend Hosting)
- MongoDB Atlas (Cloud Database)

---

## Optional Tools

- MongoDB Compass (Database visualization)
- Postman (API testing)
